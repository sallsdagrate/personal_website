# personal_website

Static GitHub Pages site wrapping the latest CV PDF with:

- embedded PDF viewer
- direct download/open actions
- recruiter-friendly intro and contact links
- live public GitHub project cards
- configurable theme color and default theme

## Update the CV

Replace `assets/cv/current-cv.pdf` with the new file.

If you want the intro, links, highlight bullets, accent color, or GitHub username to change, edit `site.config.js`.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages from the default branch root.
3. The site will serve directly from the static files in this repo.
